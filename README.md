# NeuroGrip

NeuroGrip is a web application designed for physical therapists and patients undergoing hand rehabilitation. It provides a platform to track progress, customize therapy programs, and gamify exercises to improve patient engagement.

## Features

- **Dashboard**: Get a quick overview of your therapy progress, including total hours, average grip strength, and goals completed.
- **Therapy Session**: Engage in gamified exercises like the "Grip Challenge" and receive AI-powered assistance for your robotic glove.
- **Patient Management**: Therapists can manage their patients, view their progress, and customize their therapy programs.
- **Adaptive AI Assistance**: The app uses AI to recommend the appropriate assistance level for the robotic glove based on the patient's grip strength and condition.
- **Settings**: Customize your profile and application preferences.

## Tech Stack

- **Framework**: Next.js
- **UI**: React, ShadCN UI, Tailwind CSS
- **AI**: Genkit
- **Styling**: Tailwind CSS
- **Icons**: Lucide React

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.

You can start by exploring the different sections of the application from the sidebar navigation.
